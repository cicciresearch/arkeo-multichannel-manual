The parameter file saves all JV parameters in a single file. It contains the same data as in each of the JV file, but summarized in a single table. The columns are the following:
- Time (Hours)
- Voc (V) FW
- Jsc (mA/cm2) FW
- V_MPP (V) FW
- J_MPP (mA/cm2) FW
- P_MPP (mW/cm2) FW
- Rs (Ohm) FW
- R// (Ohm) FW
- Fill Factor (%) FW
- Efficiency (%) FW
- Voc (V) RV
- Jsc (mA/cm2) RV
- V_MPP (V) RV
- J_MPP (mA/cm2) RV
- P_MPP (mW/cm2) RV
- Rs (Ohm) RV
- R// (Ohm) RV
- Fill Factor (%) RV
- Efficiency (%) RV
